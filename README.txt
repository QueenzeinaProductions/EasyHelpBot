ISTRUZIONI DEL BOT

/start                  avvia il bot
/add                    aggiunge il numero
/del                    elimina il numero
/edit                   modifica il numero
/update                 salva la configurazione/reset resetta la configurazione
/set                    imposta il messaggio automatico
/unset <NUMERO>         rimuove il messaggio automatico
/show                   mostra la lista numerata di messaggi automatici
/admin <COMANDO> ...    per la gestione degli amministratori
        COMANDI:
        register <PASSWORD>                         registra l\'utente come amministratore
        passwd <VECCHIAPASSWORD> <NUOVAPASSWORD>    cambia password
        remove <USERNAME>                           per rimuovere l\'amministratore
        show                                        per mostrare gli amministratori'
        telephone                                   per cambiare il numero di telefono dell\'amministratore
